# SSAFY_MINI_SPRING
---

## 프로젝트 목적 <br>
이 프로젝트는 SSAFY 9기 비전공 자바 5반에서 진행하는 프로젝트입니다. 교육과정을 진행하면서 웹 기능에 꼭 필요하지만 교육과정에 포함되지 않은 내용들이 무엇이 있는지 알아보고, 교육생 스스로 구현해보는 것을 목표로 합니다.  <br>
이런 저런 기능들이 있다는 사실을 알아가면서 개발자로써 시야를 더 넓게 바라보고자 했습니다. <br>
